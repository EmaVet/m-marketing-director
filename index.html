Why GitHub

⚡

The mission is real

Open source is a public good. GitHub's Social Impact team funds developers building emergency alert systems and refugee camp mapping tools. Verified nonprofits get free access to the platform. I only take roles at companies whose values I can stand behind. This is one of them.

🌍

Community at the biggest scale

I've spent my career building communities around products people trust — nonprofits, creative businesses, partners. GitHub runs that model for 180 million developers. The chance to do that work at this scale is the draw.

✦

I'm genuinely already here

I joined GitHub to teach myself AI no-code. I was mid-build when I found this role. That's not a strategy — it's just what happened. The universe had a point.

🤖

AI that protects, not extracts

GitHub has committed to protecting developer IP in AI-generated code — at a moment when most companies are going the other way. That's a values stance, not just a product decision. It matters to me.

Why I'm applying

I didn't expect to end up on GitHub. I'm not a developer. But a few weeks ago I was signing up because I needed it to publish what I was building. Then I found this role. If that's happening to me, it's happening to a lot of people — and I'm wondering how GitHub is preparing for that.
